API INTEGRATION COMPLIANCE


            

                
Operator Authorization Header

                

                

                    

                        
HTTP REQUEST STRUCTURE

                        

                            POST
                            /api/v1/webhook
                        


                        

                            Smart Payload Detection:

                            This API endpoint supports auto-detection. You do not need to specify if the target is a hash or an address. You can send requests using two formats:
                            
    
                                
  • Method 1 (JSON): Send an object with a data, tx_hash, or address key.
    • 
                                
  • Method 2 (Raw Text): Send the raw signature as plain text without any formatting.
    • 
                            

                        


                        

                            
                            
                            
                        


                        

                            
# Using Raw Text (text/plain)
curl -X POST https://your-domain.com/api/v1/webhook \
  -H "key: your_api_key_here" \
  -H "Content-Type: text/plain" \
  -d "0x8e459852f1033e9554b13625b74c44554962af1db0ed642b"

                        

                        

                            
// Using JSON (application/json)
const response = await fetch('https://your-domain.com/api/v1/webhook', {
    method: 'POST',
    headers: { 
        'key': 'your_api_key_here', 
        'Content-Type': 'application/json' 
    },
    body: JSON.stringify({ data: '0x8e459852f...' })
});

                        

                        

                            
# Using JSON (application/json)
import requests

headers = {"key": "your_api_key_here", "Content-Type": "application/json"}
payload = {"data": "0x8e459852f1033e9554b13625b74c44554962af1db"}

response = requests.post("https://your-domain.com/api/v1/webhook", json=payload, headers=headers)
print(response.json())
